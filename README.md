# 🏥 HealthGuard — Install as a Phone App

HealthGuard is a Progressive Web App (PWA) — installs on your phone home screen, runs full-screen like a native app, works offline.

## 📲 Install on Android (Chrome)
1. Run: python3 serve.py  (or double-click START_SERVER.bat on Windows)
2. Open the printed URL on your phone in Chrome (e.g. http://192.168.1.5:8080)
3. Tap "Add to Home Screen" banner OR tap ⋮ → Add to Home Screen → Install

## 🍎 Install on iPhone (Safari)
1. Run the server as above
2. Open the URL in Safari (must be Safari, not Chrome)
3. Tap Share (□↑) → "Add to Home Screen" → Add

## 💻 Just open locally (no install)
Open index.html directly in any browser. All features work except offline cache.

## Features as installed app
- Home screen icon ✅
- Full-screen, no browser bar ✅
- Works fully offline ✅
- Push notifications ✅
- Real-time alerts between patient & caregiver ✅
